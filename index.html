pysh

Python.
& Bash.

How about merging the power of Python with the powerful tools available in a shell ? If you are constantly switching from bash to python and vice-versa, look no further, you have found your nirvana!

pysh on Github Download pysh

run Python...

> import os
> for file in os.listdir():
.     with open(file) as f:
.         print(f.read())

pysh can act as a regular python interactive shell. You can do anything you would do normally with python.

or Bash...

> git status
> cat pysh | grep sys
> vi pysh
> git ci -a -m "Did changes"

Some elements of the Bash syntax can be used in pysh. Currently only the pipe construction works, but more is to come :)

or both :)
