Getting started

Download the latest version of pysh, unzip it into the directory of your choice and run ./bin/pysh.

Make sure you have Python 3 installed on your machine.

$ unzip aspyct-pysh-<version>.zip
$ cd aspyct-pysh-<version>
$ chmod +x bin/pysh
$ ./bin/pysh

Key features

  • Full support for Python
  • Partial support of Bash syntax, will be improved in the future
  • Automate your tasks with custom scripts: pysh myscript.pysh
  • Bash-like tab completion for files and folders
 

Requirements

pysh requires only Python 3 to be installed on your system, and is tested under Linux and Mac OSX.

Since pysh is a young product, it may be wiser to keep a good ol' bash as you default shell by now.

But you probably can do most of your work with pysh already :)
